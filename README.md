# twitter-madlibs-generator

This project contains the source code for a madlibs generator based on Twitter tweets.

## Install

- Get dependencies. `npm install`

## Develop

- Start it. `npm start`
- Go to [http://localhost:1729](http://localhost:1729) in your browser
